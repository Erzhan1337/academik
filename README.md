# Academik.kz

Веб-платформа для выбора магистратуры и подготовки к поступлению в Казахстане.

## Стек

- **Next.js 15** (App Router)
- **Motion** (Framer Motion v11) — анимации
- **Tailwind CSS** — стилизация
- **TypeScript** — типизация
- **Lucide React** — иконки

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Запустить в режиме разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Структура страниц

| URL | Страница |
|-----|----------|
| `/` | Главная страница |
| `/programs` | Каталог программ с фильтрами |
| `/programs/[id]` | Детальная страница программы |
| `/bolashak` | Гид по программе Болашак и проверочный тест |
| `/compare` | Сравнение программ |
| `/checklist` | Чек-лист документов |

## Функции

- ✅ Поиск и фильтрация программ (город, стоимость, язык, Болашак, направление)
- ✅ Карточки программ с анимацией (Motion layout animations)
- ✅ Плавающая панель сравнения
- ✅ Сравнительная таблица с подсветкой различий
- ✅ Интерактивный чек-лист документов с прогресс-баром
- ✅ Проверочный тест по программе Болашак (4 вопроса)
- ✅ Модальное окно напоминания о сроке подачи
- ✅ Анимированные счётчики статистики
- ✅ Адаптивный дизайн (mobile-first)
- ✅ Тёмная тема и адаптивная навигация
- ✅ Плавные page transitions
